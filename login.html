Skademaskinen : Skademanden's Remote Network Management System

This is a system to manage my servers and my home network, it is isolated such that it can only perform the actions that i have programmed in the backend.
The system is capable of booting another system and to show the last 10 lines in journalctl for a specified service for a specified device.
As a learning exercise, the backend has encrypted all personal information such that not even i can access the data without interfacing with it through this website!
